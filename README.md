#### 如何使用此脚本

**本仓库下载：[docker-installer.sh](http://gitlab.philogag.com/philogag/MyDockers/raw/master/Scripts/docker-installer.sh)**

```bash
#!/bin/sh
chmod +x ./docker-installer.sh
sudo bash ./docker-installer.sh --no-ask
```
